# シンプルなToDoリスト

ブラウザで使用できるシンプルで使いやすいToDoリストアプリケーションです。

## 機能

✨ **主な機能：**
- ✅ タスクの追加・削除
- ✅ タスクの完了状態の切り替え
- ✅ ローカルストレージを使用した自動保存
- ✅ 完了したタスクの一括削除
- ✅ タスク数の統計表示
- ✅ レスポンシブデザイン
- ✅ XSS対策済み

## 使用方法

1. `index.html`をブラウザで開く

```bash
# または簡単なHTTPサーバーで起動
python3 -m http.server 8000
```

2. テキストボックスに新しいタスクを入力
3. 「追加」ボタンをクリック（またはEnterキーを押す）
4. チェックボックスでタスクの完了状態を切り替え
5. 「削除」ボタンでタスクを削除

## ファイル構成

```
├── index.html      # HTMLファイル（UI構造）
├── style.css       # CSSファイル（スタイル）
├── script.js       # JavaScriptファイル（機能）
└── README.md       # このファイル
```
