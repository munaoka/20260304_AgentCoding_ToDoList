## 🔑 OpenAI APIキー取得方法

### 必要な手順:
1. **OpenAI公式サイトにアクセス** 
   - https://platform.openai.com

2. **アカウント作成/ログイン**

3. **API Keys ページで新規キーを生成**
   - ダッシュボード → API keys → Create new secret key
   - キーをコピー保存（`sk-...` 形式）

4. **支払い情報を登録**
   - Settings → Billing → Add payment method
   - クレジットカード情報を入力

5. **利用額制限を設定（推奨）**
   - Settings → Billing → Usage limits
   - 月額上限を設定（例：$10）

### 料金:
- **無料トライアル**: $5 クレジット（3ヶ月有効）
- **従量課金**: gpt-3.5-turbo は $0.0005/1Kトークン
- **概算**: 1日10回解析 → 月額 $0.03~0.30
